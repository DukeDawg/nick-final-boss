# Nick: The Final Boss of Minor Inconveniences

An affectionate anime-style roast site about Nick: master of smedium shirts,
unanswered texts, cinematic financial logic, and the loudest mentor arc of all
time.

## Run locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm run build
```

Pushes to `main` publish automatically through GitHub Pages.
